🛠 Setup Guide
1
Download Client Portal Gateway
Download from IBKR API page → "Client Portal API". Unzip to ~/clientportal
2
Start the Gateway
In Terminal:
cd ~/clientportal && bin/run.sh root/conf.yaml
3
Authenticate
Open https://localhost:5000 in Safari → accept self-signed cert → log in with your IBKR paper account credentials.
4
Update Python Proxy
Replace the proxy file with the updated version that adds IBKR routes. See OptionsPro_proxy.py from this session.
cp ~/Desktop/OptionsPro_proxy.py /Applications/OptionsPro.app/Contents/MacOS/OptionsPro
⚠ This replaces the proxy binary — make sure Python 3 is installed on your Mac.
5
Connect
Click Connect above. The proxy on port 5010 forwards requests to the gateway on port 5000. Green dot = ready.
